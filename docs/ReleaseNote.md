# 版本发布记录

## 2025-11-15 v1.0.1
- 增加策略查看功能，实现自动注册策略
  - 支持查看交易策略
  - 支持查看信号策略

## 2025-11-12 v1.0.0
- 初始版本发布
    - 数据获取
      - 支持从csv读取、
      - 支持从指定数据源获取
        - akshare
        - baostock
        - futu，需要申请账号才能使用sdk
    - 执行回测
      - 单股票回测
        - 支持指定回测数据
        - 支持指定初始资金
        - 支持指定交易手续费（settings文件中设置）
        - 支持指定交易滑点（settings文件中设置）
      - 多股票批量回测
        - 支持指定回测数据
        - 支持指定初始资金
        - 支持指定交易手续费（settings文件中设置）
        - 支持指定交易滑点（settings文件中设置）
      - 历史回测结果查看、可视化图表
        - 股价、成交量等基础数据
        - 交易记录
        - 持仓记录
        - 交易信号
        - 总资产变化
      - 交易信号可视化图表
        - 按照信号类型聚合，支持按照时间、股票、信号类型进行筛选
        - 支持导出HTML
