# 版本发布记录

## 2026-04-16 v2.0.0

### 新增功能

#### 前端界面重构
- 全新的现代化UI界面，深色主题设计
- 统一的样式系统，CSS样式从HTML文件中抽离

#### 信号分析页面
- 新增信号分析功能页面
- 支持多维度筛选（时间、股票、策略、信号类型）
- 信号分布统计图表
- 支持批量选择信号文件分析

#### 背景音乐播放器
- 新增独立音乐播放器窗口
- 支持播放列表管理
- 音量控制

#### 定时任务执行记录
- 新增任务执行历史记录功能
- 支持分页展示
- 记录执行状态、耗时、处理结果
- 实时刷新执行进度

#### 企业微信通知
- 新增 Webhook 通知功能
- 定时任务完成后自动发送报告到企业微信群

### 架构优化

#### 模块化重构
- 新增 `core/ai/` 模块 - AI分析功能
- 新增 `core/notification/` 模块 - 通知功能
- 新增 `core/signal/` 模块 - 信号处理功能
- 新增 `core/task/` 模块 - 定时任务功能
- 策略模块拆分为 indicator 和 trading 两个子模块

#### 可视化增强
- 使用 Plotly 重新实现图表渲染
- 支持交互式图表操作
- 优化图表加载性能

### 依赖更新
- Python 3.13 支持
- 新增 requirements-13.txt

## 2025-11-12 v1.0.0

      - baostock
      - futu，需要申请账号才能使用sdk

    - 单股票回测

      - 支持指定初始资金
      - 支持指定交易手续费（settings文件中设置）
      - 支持指定交易滑点（settings文件中设置）
    - 多股票批量回测
      - 支持指定回测数据
      - 支持指定初始资金
      - 支持指定交易手续费（settings文件中设置）
      - 支持指定交易滑点（settings文件中设置）
    - 历史回测结果查看、可视化图表
      - 股价、成交量等基础数据
      - 交易记录
      - 持仓记录
      - 交易信号
      - 总资产变化
    - 交易信号可视化图表
      - 按照信号类型聚合，支持按照时间、股票、信号类型进行筛选
      - 支持导出HTML
